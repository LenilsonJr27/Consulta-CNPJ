# Consulta CNPJ (BrasilAPI)

Aplicação simples para consultar dados de empresas a partir do **CNPJ**, consumindo a **BrasilAPI**.  
Feita com **HTML + CSS + JavaScript**, utilizando **Fetch API**, validação do CNPJ (14 dígitos) e tratamento de erros.

##  Funcionalidades

- Consultar CNPJ na BrasilAPI
- Validação do input (somente números e 14 dígitos)
- Exibição organizada dos dados retornados
- Tratamento de erros (CNPJ inválido, HTTP error, falha de rede)
- Exibição do JSON bruto para debug (opcional)

##  Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- [BrasilAPI](https://brasilapi.com.br/)
